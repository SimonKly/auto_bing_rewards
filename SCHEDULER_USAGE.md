# Auto Bing Rewards 定时任务使用说明

## 功能描述
此项目支持使用APScheduler定时执行必应奖励任务：
- 每天凌晨4点自动执行一次任务
- 如果4点执行失败，则在当天上午8点再次尝试执行

## 安装依赖
在运行定时任务前，请确保已安装所有依赖：

```bash
pip install -r requirements.txt
playwright install
```

## 运行定时任务
要启动定时任务，请运行以下命令：

```bash
python run_scheduler.py
```

程序将持续运行并按计划执行任务。

## 配置说明
- 任务会在每天凌晨4点准时执行
- 如果4点的任务执行失败，系统将在上午8点再次尝试
- 所有执行日志都会记录在日志文件中

## 注意事项
- 确保系统时间准确
- 程序需要保持运行状态才能执行定时任务
- 如果4点的任务成功执行，8点的任务将自动跳过
- 如果4点的任务失败，8点会自动重试
